# 🚖 Taxsi Subscription System

**Taxsi Subscription System** is a mobile application designed to help taxi drivers manage subscription-based services, track payments, and monitor financial activity بسهولة وبدون تعقيد.

The app simplifies the process of handling subscriptions, making it easier for drivers to stay organized and maintain consistent income tracking.

---

## 📌 Overview

Managing subscriptions and payments manually can be difficult and time-consuming.  
**Taxsi Subscription System** provides a digital solution that allows drivers to:

- Track active subscriptions  
- Manage payments بسهولة  
- Monitor income over time  
- Keep records organized  

---

## ✨ Features

- 🚕 Manage taxi subscription plans  
- 💳 Track payments and dues  
- 📊 View financial summaries  
- 🧮 Calculate total income  
- 📝 Simple and clean interface  
- ⚡ Fast and responsive performance

## ⚙️ Installation

1. Clone the repository:

git clone https://github.com/issamqabha/taxsi_sub.git

2. Navigate to the project directory:

cd taxsi_sub

3. Install dependencies:

flutter pub get

4. Run the app:

flutter run
